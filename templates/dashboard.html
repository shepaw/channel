接入文档

1. 获取 Access Token

在 "Access Token" 页面生成临时 Token（默认有效期 15 分钟）。

2. 注册 Channel

POST /api/v1/channels
Authorization: Bearer <your_token>
Content-Type: application/json

{
  "name": "My App Channel",
  "type": "http",          // http | https | ws | tcp | udp
  "target": "http://localhost:3000",
  "description": "My application"
}

返回值示例：

{
  "id": "3f8a2b1c-...",
  "name": "My App Channel",
  "type": "http",
  "endpoint": "https://channel.example.com/proxy/3f8a2b1c-...",
  "is_active": true
}

3. 通过 Channel 访问应用

# HTTP/HTTPS 转发
GET https://channel.example.com/proxy/{channel_id}/your/path

# WebSocket 连接
ws://channel.example.com/proxy/{channel_id}/ws

# TCP/UDP (返回的endpoint中已包含端口信息)
tcp://channel.example.com:{port}

4. 设置限流规则

POST /api/v1/channels/{id}/rate-limits
Authorization: Bearer <your_token>

// 带宽限制（单位 Mbps）
{"rule_type": "bandwidth", "limit_value": 10, "time_window": "1s"}

// 并发连接数限制
{"rule_type": "connections", "limit_value": 100, "time_window": "1m"}

// 请求频率限制（次/秒）
{"rule_type": "requests", "limit_value": 100, "time_window": "1s"}

创建 Channel

🔗 隧道模式说明

创建后，在本地运行以下命令建立连接：

./channel-agent --server http://localhost:8080 --channel-id <创建后获取> --token <your-token> --target http://localhost:3000

外部流量将通过此隧道转发到你的本地应用，无需公网 IP。
